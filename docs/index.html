🍽️ MealHunter

Android-приложение для поиска и сохранения рецептов на основе Spoonacular API.

Требования Архитектура База данных Спецификация Презентация

О проекте

MealHunter позволяет листать рецепты как карточки в стиле Tinder, искать по названию и ингредиентам, сохранять избранное и фильтровать по кухне, диете и времени готовки.

  • Минимальная версия Android: 8.0 (API 26)
  • Целевая версия Android: 14 (API 34)
  • Язык: Kotlin
  • UI: Jetpack Compose + Material 3

  2. Добавить SPOONACULAR_API_KEY=ваш_ключ в файл gradle.properties
  3. Открыть в Android Studio и запустить на эмуляторе или устройстве
