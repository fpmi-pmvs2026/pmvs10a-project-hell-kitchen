# Дополнительная спецификация

## Ограничения

| Параметр | Значение |
|----------|---------|
| Минимальная версия Android | 8.0 (API 26) |
| Целевая версия Android | 14 (API 34) |
| Таймаут сетевых запросов | 30 секунд |
| Максимум записей в истории поиска | 20 |
| Лимит бесплатного Spoonacular API | 150 запросов в сутки |

## Безопасность

- API-ключ **никогда не хранится в исходном коде** — только в `gradle.properties` (в `.gitignore`) и GitHub Secrets
- Ключ внедряется на этапе сборки через `BuildConfig.SPOONACULAR_API_KEY`
- Никакие пользовательские учётные данные не собираются и не хранятся
- Все сетевые запросы выполняются по HTTPS к `api.spoonacular.com`

## Надёжность

- Сетевые ошибки перехватываются в Repository и возвращаются как `Result.failure` — UI отображает кнопку повтора
- База данных Room использует `fallbackToDestructiveMigration` при разработке; в продакшене необходимы явные миграции
- Загрузка изображений через Coil обрабатывает ошибки без сбоев (показывает плейсхолдер)
- Избранное доступно **офлайн** — хранится локально в Room, сеть не требуется

```
┌──────────────────┐

         │ (при успехе)

┌──────────────────┐

         │
         ▼
┌──────────────────┐

│ (артефакт)       │  хранение: 7 дней
└──────────────────┘
```
